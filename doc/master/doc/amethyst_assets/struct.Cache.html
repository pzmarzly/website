pub struct Cache<T> { /* fields omitted */ }

A basic implementation for a cache. This might be useful as the Context of an Asset, so that the same asset doesn't get imported twice.

Because contexts have to be immutable, a RwLock is used. Therefore, all operations are blocking (but shouldn't block for a long time).
